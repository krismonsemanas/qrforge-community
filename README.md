# QR Forge Community

Public community hub for QR Forge.

QR Forge is a modern privacy-first QR code toolkit built for speed, simplicity, and beautiful customization.

## Core Principles

- Fully client-side
- Privacy-first
- No login required
- No watermark
- Free forever
- Fast and lightweight
- Mobile-first
- PWA-ready

## Website

- Website: https://getqrforge.site

## Purpose of This Repository

This repository exists for:

- bug reports
- feature requests
- roadmap discussions
- community feedback
- QR template ideas
- public changelog tracking

QR Forge itself is currently closed-source.

This repository is intended to keep development transparent while maintaining a focused and sustainable product direction.
