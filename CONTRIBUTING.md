# Contributing

Thank you for your interest in QR Forge.

This repository is primarily used for:

- bug reports
- feature requests
- roadmap discussions
- community feedback

QR Forge source code is currently private.

---

## How You Can Contribute

### Report Bugs

Please include:
- browser/device information
- reproduction steps
- screenshots if applicable

### Request Features

Clearly explain:
- the problem
- the proposed improvement
- the use case

### Share Ideas

Use GitHub Discussions for:
- feature ideas
- QR design inspiration
- workflow suggestions
- showcase discussions

---

## Philosophy

QR Forge values:
- simplicity
- performance
- privacy
- accessibility
- thoughtful UX

Please keep discussions constructive and focused.
